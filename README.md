# Daniel Brown Portfolio

## Files
- `index.html` — professional Bootstrap portfolio and HTML résumé
- `scratch.html` — from-scratch assignment page
- `css/styles.css` — professional-site styling
- `css/scratch.css` — scratch-page styling created from scratch
- `js/scripts.js` — smooth navigation
- `assets/img/profile.svg` — placeholder profile image
- `assets/img/election-illustration.svg` — original scratch-page image

## Use in VS Code
1. Copy these files into your website folder.
2. Open `index.html` with Live Server.
3. Replace `assets/img/profile.svg` later with your own image if desired.
4. Review every résumé statement for accuracy before publishing.

## GitHub Pages
Upload the contents of this folder to the root of the GitHub repository. Then open:
Settings → Pages → Deploy from a branch → main → /(root).

## Privacy
The public website intentionally omits the full street address and phone number. Add them only if you are comfortable displaying them publicly.
